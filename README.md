# Support Ticket Classification & Prioritization System

## Project Overview
This project automatically classifies customer support tickets into categories and predicts ticket priority levels using Natural Language Processing (NLP) and Machine Learning.

## Features
- Text Cleaning and Preprocessing
- Tokenization and Stopword Removal
- TF-IDF Feature Extraction
- Ticket Category Classification
- Ticket Priority Prediction
- Model Evaluation
- Confusion Matrix Visualization

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Matplotlib
- Seaborn
- Joblib
- Jupyter Notebook

## Dataset
Customer Support Ticket Dataset

## Machine Learning Workflow
1. Load Dataset
2. Clean Ticket Text
3. Convert Text to TF-IDF Features
4. Train Logistic Regression Models
5. Predict Ticket Type
6. Predict Ticket Priority
7. Evaluate Performance

## Results

### Ticket Type Classification
Accuracy: 20.1%

### Priority Prediction
Accuracy: 26.3%
